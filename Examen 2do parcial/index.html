    Explain to a children what is javascript?

    Javascript es un lenguaje de programación para para hacer scripts en internet. Lo que Javascript hace es comunicarse con el navegador de internet y le dice que hacer. Mostrar imágenes, colores, botones, información. Se comunica con otro lenguaje llamado HTML que en conjunto lo que hacen es que uno proporciona información para que se vea en tu navegador de internet mientras que Javascript en esencia hace que la página se vea de manera elegante y bonita, con estilo podría decirse.

    What is Jquery?

    JQuery es loque podría decirse como una extensión de Javascript, lo complementa con másfunciones de utilidad y fácilita el trabajo a la hora de interpretar lo queJavascript quiere decir como código. Es en esencia una librería que agrega máscosas que puede hacer javascript en el navegador de internet.
